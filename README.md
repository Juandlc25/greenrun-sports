### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Juandlc25/greenrun-sports
   ```
2. Locate in the project folder

   ```sh
   cd greenrun-sports/
   ```

3. Install project dependencies

   ```sh
   npm install
   ```
